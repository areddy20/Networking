
Pre-requisite: Install following package using pip3 package manager (Program uses external library dnslib)
------------------------------------------------------------------------------------------

pip3 install dnslib


Python version is python3.x

How to run program as follows:
------------------------------

python3 my-dns-client.py gmu.edu
python3 my-dns-client.py www.google.com
python3 my-dns-client.py www.amazon.com
python3 my-dns-client.py www.cnn.com


If you don't specify any hostname, program should display usage syntax
